# ArcGIS Skeleton Starter

This repo is a starter for creating ArcGIS JS API v4 webpages.

## Includes
- Types (ArcGIS, Dojo, Typescript)
- Jest Testing Framework
- Tailwind CSS

## Requirements
- Node & NPM
- Typescript

## Setup
- Clone / copy repository
- Run ```yarn```

## Building
- ```yarn build```
- or ```yarn watch```
## Testing
- ```yarn test```
